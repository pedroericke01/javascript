Aprendendo Javascript

Para inicializar um script em qualquer linguagem de programação na web, podemos utilizar 2 tecnicas:

  1. inset:o script será colocado abaixo de todo o conteúdo do website, dentro do arquivo html base.
  2. OutSet: o script será desenvolvido em outro arquivo externo, especifico para a programação e iterações da linguagem pela web.
Nesse exercício vamos utilizar a técnica 1 para exibir o:'olá mundo!' atravé de um alerta que o navegador vai exibir para o usuário.
